# Wind Reflexology Website

Welcome to the Wind Reflexology website! This project is a static website designed to promote the services of Wind Reflexology.

Deployed site: https://wing-reflexology1.onrender.com/
## Features

- Responsive Design: The website is designed to be responsive, adapting to different screen sizes and devices.
- Simple and Fast Navigation: The website has a simple and intuitive navigation system, allowing users to easily navigate between different sections.
- Accessibility: The website is designed to be accessible to users with disabilities, following best practices for web accessibility.
- Branding: The website's color theme matches the branding logo of Wind Reflexology, creating a cohesive and visually appealing design.
- About Section: The website has an About section that provides information about Wind Reflexology and its services.
- Services Section: The website has a Services section that lists the different services offered by Wind Reflexology.
- Contact Information: The website includes contact information, allowing users to easily get in touch with Wind Reflexology.

## Tools Used

- Next.js: The website is built using Next.js, a popular React framework for building web applications.
- React Components: The website uses various custom and third-party React components for building the UI.
- Tailwind CSS: The website is styled using Tailwind CSS, a utility-first CSS framework.
- Lucide React Icons: The website uses icons from Lucide React, a set of simple and customizable icons.

## Installation

1. Clone the repository: `git clone https://github.com/your-username/wind-reflexology-website.git`
2. Install dependencies: `npm install`
3. Start the development server: `cd wing-reflexology && npm run dev`

## Running the Project

1. Open your browser and navigate to `http://localhost:3000`
2. Explore the website and its features
